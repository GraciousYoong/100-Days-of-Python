# 100 Days of Python

This repository contains my Python learning journey and practice projects from the **100 Days of Python** challenge.

## Requirements

Make sure you have Python installed.

You can download Python here:

https://www.python.org/downloads/

To check if Python is installed:

```bash
python --version
```


---

## Clone This Repository

Clone the repository to your computer:

```bash
git clone https://github.com/GraciousYoong/100-Days-of-Python.git
```

Go into the project folder:

```bash
cd 100-Days-of-Python
```

---

## How to Run the Files

Run a Python file using:

```bash
python <file_name>.py
```

Example:

```bash
python Day1_band_name_generator.py
```

Another example:

```bash
python Day4_rock_paper_scissors.py
```
